# Graphic-Intersection
## 安装说明
### Mac端 安装Vue3环境
1. 下载Node.js
[官网链接](https://nodejs.org/en/download)
2. 检验是否安装成功
  node -v
  npm -v
3. 安装Vue
  sudo npm install webpack -g
4. 安装Vue CLI（脚手架）
  sudo npm install -g @vue/cli
5. 运行Graphic-Intersection项目
（1）进入Graphic-Intersection项目
  cd Graphic-Intersection
（2）npm run serve
